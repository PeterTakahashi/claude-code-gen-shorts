このプロジェクトは何か

Claude Code に日本語で頼むだけで、
ショート動画が 生成 → アップロード されるリポジトリ。

  • 絵を描く・音声を作る・動画を合成する・YouTube に上げる — 全部 Claude がやる
  • 人がやるのは テーマを決めて、シナリオに OK を出す だけ
  • そして —— ソースを自由に書き換えて、「動画の作り方」自体をみんなが変えられる のが目的

Claude Code とは

Anthropic の、ターミナルで動く AIエージェント。チャットと違うのは —

  • ファイル編集・コマンド実行・web検索・git操作を 自分で判断して連続実行
  • 「コードを補完する」ではなく 「タスクを丸ごと任せる」 道具
  • だから「動画を1本作ってアップして」みたいな 長い仕事を一気に通せる

料金は Claude のサブスク (Pro $20 / Max $100〜) に込み。今日はこれ1つで全部やる。

誰が何をやるのか

👤 人がやること（これだけ）

  • ① 最初に Claude Code を入れる
  • ② 認証まわりの初期設定
    APIキー / YouTube・GCP の OAuth
  • ③ テーマを決めて、シナリオに OK を出す

🤖 Claude がやること（頼むだけ）

  • シナリオを書く
  • イラストを生成する
  • ナレーション音声を作る
  • 字幕・効果音つきで動画に合成する
  • YouTube にアップロードする
  • 再生数を集計して分析できる状態にする

手を動かす必要があるのは 「最初の導入」と「認証」 だけ。あとは会話で回る。

セットアップ ①　まず動かす

# 1. Claude Code を入れる（人がやる初回作業）
npm i -g @anthropic-ai/claude-code

# 2. このリポジトリを取ってくる
git clone https://github.com/PeterTakahashi/claude-code-gen-shorts
cd claude-code-gen-shorts

# 3. Claude Code を起動（初回はブラウザでログイン）
claude

起動したら、あとは Claude に日本語で頼むだけ。
依存パッケージのインストールも「環境を整えて」と頼めばやってくれる。

セットアップ ②　画像と音声

  • 画像GEMINI_API_KEY を取得して .env に書く
    aistudio.google.com で無料発行 → .env.example をコピーして .env を作り貼るだけ
  • 音声Aivis Speech（日本語TTS・無料）をローカルで起動
    アプリを立ち上げておくと、Claude がそこに喋らせて音声を作る
  • 指示書Skill は同梱済み。/biography-webtoon のように / で呼べて、ジャンルごとの作り方を Claude が読み込む

ここまでで「動画を作る」ところまでは動く。アップロードしたい人だけ次へ。

セットアップ ③　YouTube に上げる（認証）

一番手間なのがここ。でも 最初に1回だけ。

  1. Google Cloud で YouTube Data API v3 を有効化
  2. OAuth クライアントの認証情報を作って、リポジトリに置く
  3. アップ先の YouTube チャンネルを作る
  4. 初回だけ ブラウザで OAuth 認証 → トークンが保存される

2回目以降は保存トークンで自動。＝ 認証が終われば、もう手作業はほぼゼロ。

動画の作り方（実際の会話）

あなた ＞ 動物雑学チャンネルに、ハチドリの面白いショートを作って

Claude ＞ シナリオ案です（6パネル構成）
          1. 掴み：ハチドリは1秒に50回羽ばたく…
          2. …
          6. オチ：実は…
          これで進めていい？

あなた ＞ OK。3番をもっと意外な事実に変えて

Claude ＞ 修正しました。では生成します
          → イラスト6枚 生成
          → ナレーション音声 合成
          → 字幕・効果音つきで動画に合成
          → YouTube にアップロード 完了
          https://youtube.com/shorts/xxxxxxxx

人の操作は 「作って」 と 「OK」 の2回だけ。

1日の運用ループ

  1. 朝、ダッシュボードで 伸びてるトピック を眺める
  2. 今日のネタを 1〜2 個 人が決める
  3. Claude に「このネタでショート作って」と一行投げる
  4. シナリオを確認して OK → 生成・アップまで自動
  5. 夕方にもう 1〜2 本

実働 30分/日 × 1ヶ月で、累計 339k 再生まで来た。

このリポジトリの目的

「動画の作り方」そのものを、各自が書き換えて使う。

  • 絵柄・尺・構成・字幕ルール — 指示書（Skill）を書き換えれば変わる
  • 新ジャンル — Skill を1つ足せば、新しいチャンネルが立ち上がる
  • 画像・音声エンジン — 別のものに差し替えてもいい
  • 気に入らない挙動は Claude に「ここをこう直して」と言えばソースごと変わる

完成品を配るのではなく、みんなが自分用に改造する土台として置いてあります。

ハンズオン（各自1本）

  1. git clone → claude 起動
  2. .env に GEMINI_API_KEY を入れる / Aivis を起動
  3. 「○○のショートを作って」と頼む（アップは認証済みの人だけ）
  4. シナリオに OK → 出力された動画を開いて確認
  5. 余裕があれば「もっと絵を派手にして」など、好みに改造してみる

詰まったらその場で Claude に直接聞くのが一番早い。
