# YouTube Shorts ジャンル戦略 — 2026年版

## TL;DR

**今すぐ立ち上げ推奨3チャンネル** (既存パイプライン + 1人=1ジャンル + 各2本/日):

| 優先 | チャンネル | 既存資産活用度 | RPM見込 | 飽和度 |
|---|---|---|---|---|
| 1 | **注目スタートアップ (JP)** | ◎ biography枠そのまま | 中-高 | 低 |
| 2 | **物理学トリビア (JP)** | ◯ 同パイプ流用 | 中-高 (education) | 中 |
| 3 | **恋愛心理テクニック (JP)** | ◯ 同パイプ流用 | 中 | 高だが anime差別化可 |

既存:
- `ijinden_ja` (@baltoon-ai) — 偉人伝
- `baltoon_biography_en` (@BaltoonBiography) — 英語版偉人伝
- `baltoon_incidents_ja` (バルトゥーン事件紹介) — 企業事件

新規3chと合わせて **計6ch / 日合計12本** のオペレーションが現実的ライン。

---

## 1. 2026年マクロ前提

### 「Inauthentic Content」規制 (最重要)
- 2025年7月「Repetitious content」 → 2026年1月「Inauthentic content」に改称・厳格化
- ターゲット: 「同じテンプレートで日20本量産」「AI生成スライド + TTS で記事を読み上げ」「名詞を入れ替えて50本」
- **死に筋**: 「テンプレ × 大量投下」
- **生存策**: **scriptの個別性 (curiosity-gap hook + 内容のバリエーション)** ← 既存biography成功パターンが既に合致
- 量より「同一テンプレ」がトリガー → 1ch×2本/日は安全圏、ただし script の hook と中身が毎回ユニークである必要

### 2026年のShorts RPM
| 区分 | RPM目安 |
|---|---|
| 一般 | $0.03–$0.10 |
| Finance / B2B | $0.15–$0.45 |
| Education (科学・物理) | $0.08–$0.20 |
| JP fee-ad CPM | ¥600–¥1,800 |

Shortsは**ファネル** — 直接収益は薄い。長尺spilloverまたはチャンネル登録 → 長尺視聴で本格収益化。

### YMYL ペナルティ
金融・医療・健康系は**「これは投資助言ではありません」表示が冒頭10秒で必須**。違反 → 自動 demonetize。

## 2. ジャンル別評価

### A-tier (最優先で着手すべき)

#### A1. 注目のスタートアップ
**概要**: 「Company X が $Y 調達」「創業者 Z の意外な経歴」を 30-45秒 anime + TTS で。
- **demand**: JPはTechCrunch Japan、BRIDGE、Coral Capital 等が text媒体で独占、Shorts形式は空白
- **RPM**: B2B/tech枠で Shorts $0.15-$0.45、長尺 $10-$25
- **pipeline適合度**: ◎ — biography skill のテンプレを流用、人物 description だけ差替え
- **政策リスク**: ほぼゼロ (factualニュース)
- **既存viral pattern適合**: 高 (「23歳が$10億断った日」型と同型)
- **content supply**: 無尽蔵 (毎週新規スタートアップ・ファンド・買収)
- **チャンネル名候補**: 「バルトゥーン startup」/「3分でスタートアップ」/「資金調達ニュース AI」

#### A2. 物理学トリビア
**概要**: 「光速で動いたら時間はどうなる?」「もし地球が立方体だったら?」型のwhat-if anime
- **demand**: JP Shorts 内では surprisingly 空白 (長尺は Kurzgesagt-JP 等あるがShortsは未開拓)
- **RPM**: Education tier $0.08-$0.20 Shorts、長尺$10-$25
- **pipeline適合度**: ◯ — キャラ不要、抽象シーン描けばOK
- **政策リスク**: 最低 (family-safe)
- **既存viral pattern適合**: 中-高 (curiosity gap型 hookが書きやすい)
- **content supply**: 大学物理入門レベルで100ネタ以上
- **チャンネル名候補**: 「物理アニメちゃん」/「もしも物理学」/「30秒物理」

#### A3. 恋愛心理テクニック
**概要**: 「男性が好き避けする5つのサイン」「会話3秒目で好感度が決まる理由」等 anime寸劇 + TTS
- **demand**: エンタメ = JP Shorts 視聴の約70%、恋愛心理サブピラーは飽和ぎみだが anime形式は差別化余地
- **RPM**: $0.05-$0.15、ブランドセーフ
- **pipeline適合度**: ◯ — 男女2キャラ + シチュエーション
- **政策リスク**: 低-中 (「絶対モテる」式断定 + ジェンダー対立煽りは要回避)
- **既存viral pattern適合**: 高 (「○○の本当の理由」型と完全一致)
- **content supply**: 数百ネタ (心理学本 + Reddit r/dating)
- **チャンネル名候補**: 「恋愛科学Anime」/「3分恋愛心理」

### B-tier (条件付き launch)

#### B1. 占い・タロット
- **demand**: 非常に高い、ただし RPM 低い (spiritual = 限定広告)
- **pipeline適合度**: ◎ (タロットカードめくり anime)
- **saturation**: 上位 (橋本京明, あんずまろん) は強いが Shorts 「今日のタロット」型は隙あり
- **risk**: 占い系は YouTube 広告システムが「supernatural claim」扱い、CPM 天井
- **判断**: views狙いには良い、収益は期待薄

#### B2. スピリチュアル (英語版)
- **demand**: 英語圏は manifestation/energy work で大きいが JP は薄い
- **risk**: 多くが demonetize される
- **判断**: i18n.en パイプ稼働中なら追加チャレンジ可能、JP単独はskip

### C/D-tier (skip推奨)

#### C1. 仮想通貨 / 株 (デイトレ・値動き)
- **risk**: YMYL 規制で「投資助言ではない」表示必須、JP FSA も厳しい
- **pipeline mismatch**: 視聴者はリアルチャート/ティッカー期待、anime panel不適合
- **incumbent**: ヤーマン, バフェット太郎 等 personality-led 強い
- **判断**: skip

#### C2. 英語学習
- **incumbent**: POC English, TAKASHii, バイリンガール 等 native-speaker authenticity 必須
- **algo bias**: 言語カテゴリは native speech 偏重
- **anime差別化**: 弱い (「anime キャラで英熟語」等のニッチ angle なら可能)
- **判断**: skip (差別化angleがある場合のみ)

## 3. 推奨ポートフォリオ

### 既存3ch + 新規3ch = 6ch並列運用

| ch | テーマ | 言語 | 日次cadence | 状態 |
|---|---|---|---|---|
| ijinden_ja | 偉人伝 | JA | 2/day | 稼働中 |
| baltoon_biography_en | 偉人伝EN | EN | 2/day | 稼働中 (本数まだ少) |
| baltoon_incidents_ja | 企業事件 | JA | 2/day | 稼働中 (3本) |
| (新) startup_news_ja | スタートアップ調達ニュース | JA | 2/day | **着手 #1** |
| (新) physics_anime_ja | 物理学 what-if | JA | 2/day | **着手 #2** |
| (新) love_psych_ja | 恋愛心理 anime寸劇 | JA | 2/day | **着手 #3** |

**1日12本upload** (6ch × 2/day) → 1ヶ月 360本 → Gemini panel生成 360×6panels = 2,160枚 ≈ Gemini batch $40-60/月

### YouTube quota 試算
- Upload 1本 = 1,600 units
- 12本/日 = 19,200 units/日
- 1 Google Cloud project quota = 10,000 units/日 (default)
- → **各チャンネルに別の Google Cloud project が必須** (= 6 projects)
- もしくは quota increase 申請

### 並行ローンチ順
1. **Week 1**: startup_news_ja の基盤構築 (bootstrap + 最初の5本生成 + 公開)
2. **Week 2**: physics_anime_ja
3. **Week 3**: love_psych_ja
4. **Week 4以降**: 各ch安定運用、stats を見てpivot判断

## 4. 各ジャンル実装メモ

### startup_news_ja
- **テンプレ**: 「[Company名] が [金額] 調達した話」「[創業者] の意外な経歴」「[Company名] が [事件/快挙] した日」
- **ネタ源**: Crunchbase, TechCrunch (英), PR Times (JP), Coral Capital blog
- **panel構成**: 創業者ポートレート / 製品 / ファンドロゴ / グラフ
- **biography skill 完全流用可能**
- **Gemini プロンプトに「Logo認識」のpolicy回避**: 「[Company名] のスタイルのロゴ」式に間接表現

### physics_anime_ja
- **テンプレ**: 「もし [現象] だったら [結果]」「[実験名] が起きた瞬間」
- **ネタ源**: Reddit r/AskPhysics, Kurzgesagt スクリプト, 物理学事典
- **panel構成**: 抽象シーン主体 — キャラ最小化、シミュレーション風
- **既存pipeline不足分**: 数式オーバーレイ機能 (現状未実装)、必要なら ffmpeg drawtext で簡易対応
- **チャンネル個性**: ナレーターを「博士キャラ」固定 → recurrence生む

### love_psych_ja
- **テンプレ**: 「男性が [行動] する3つの理由」「[シチュ] で女性が惹かれる瞬間」
- **ネタ源**: 心理学書 (DaiGo系)、海外 dating science research
- **panel構成**: 男女2キャラ + 喫茶店/オフィス/LINEトーク等のシチュ
- **既存pipeline不足分**: LINE風吹き出し描画 (Gemini で「smartphone screen with chat bubbles」prompt 可)
- **risk軽減**: タイトルに「断定」入れない (「絶対モテる」→「モテやすい人の共通点」)

## 5. 既存biography ch から学んだ実装ルール (全ch共通)

1. **Hook caption は最初の2秒で curiosity-gap 完成**
   - ✅ 「ジョブズが風呂に入らなかった本当の理由」 (2,538 views)
   - ❌ 「ジョブズ第3話 大学時代」 (汎用、視聴ゼロ)
2. **タイトル冒頭12字以内に hook 完結** (Shorts フィードpreviewが12字cut)
3. **panels の場所と国を明示** (「Cupertino California」「Tokyo Japan」)
4. **画像内日本語禁止** (多言語展開対応)
5. **Conjunction prose** (「Xだったので、Y」「Xから、Y」) で大人向け文体
6. **西暦には相対時間併記** (「30年後の2014年」)
7. **panel と narration の独立** (i18n対応で順序入れ替え可)
8. **1ch内で series_brand を統一** (channel認識)
9. **font_scale 4.0 / NotoSansJP-Regular** で hard-burned caption
10. **TTS speed 1.4倍**で punchy ペース

## 6. 投稿 cadence と algo配慮

| ルール | 詳細 |
|---|---|
| **1日2本まで/ch** | 3本目以降フィード抑制リスク (ユーザー観測) |
| **時間帯固定** | JST 19:00 と 22:00 等、algoが学習する |
| **連続2本の hook を変える** | 同テンプレ連投はpenalty |
| **同topic 連投NG** | 「Apple系」「ロケット系」を続けないでローテーション |
| **既存viral動画を削除しない** | チャンネルhealth低下 |
| **dead videos の削除は慎重に** | 全体4-7日経って views<30 → 削除可だが多数連続削除はchannel signal悪化 |

## 7. 失敗判定 + pivot基準

- **Launch 後 14日 view平均 < 100/本** → ジャンル再考
- **Launch 後 30日 channel登録 < 100** → hook 再設計 or 別ジャンル
- **「Inauthentic content」警告メール受領** → templating 緩和 (panel数 / panel構成 / hook型 を3-5パターンに増やす)
- **広告制限 yellow icon が3本以上** → 当該ジャンル不向き、別ジャンルへピボット
