# Adding a new novel

1. Copy this template:

   ```bash
   cp -r projects/_template projects/<your_novel_id>
   cd projects/<your_novel_id>
   mv project.yaml.example project.yaml
   rm characters.yaml.example  # generated by pipeline S2
   ```

2. Drop the source text at `input/novel.txt` (UTF-8 plain text). For Aozora HTML,
   pre-strip ruby/annotations or run the existing `src/novel_loader.py` against
   the HTML first.

3. Edit `project.yaml` — title, author, era, style, chapter markers.

4. Run the pipeline. It advances to the next human-review checkpoint and stops:

   ```bash
   uv run python -m src.pipeline <your_novel_id>
   ```

5. At each stop, follow the printed instructions, then re-run the same command
   to continue.

The full stage list is in `/SPEC.md` (top of the repo).
